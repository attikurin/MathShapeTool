算数・数学 図形作成ツール (Math Shape Tool)

小中学校の先生やICT支援員の方々向けの、教材作成支援Webツールです。
テスト問題や配布プリントに必要な「正確な図形」を、Webブラウザ上で簡単に作成し、画像として保存できます。

📌 特徴

インストール不要: HTMLファイルを開くだけですぐに使えます。

正確な作図: 辺の長さや角度を数値で入力すると、自動的に正しい比率で描画されます。

多彩な図形: 三角形（3辺、SAS）、四角形、円、角度、座標指定などに対応。

プリント作成に最適:

背景透過: WordやPowerPointに貼り付けても、文書の背景を邪魔しません。

方眼紙モード: 図形のサイズ感を確認しながら作業できます（画像保存時には方眼紙は消えます）。

ラベル表示: 頂点（A, B, C...）や辺の長さを自動で表示します。

🚀 使い方

math_shape_tool.html ファイルをダブルクリックして、Google ChromeやEdgeなどのブラウザで開きます。

画面左側のメニューから、作成したい図形の種類を選びます。

数値を入力して形を調整します（プレビューはリアルタイムで更新されます）。

図形をドラッグして、キャンバス内のいい位置に移動させます。

右上の 「画像保存 (PNG)」 ボタンを押して、画像をダウンロードします。

ダウンロードした画像を、WordやPowerPointなどに貼り付けて使用してください。

🛠 技術情報

HTML5 (Canvas API)

Tailwind CSS (CDN)

Single File Component (1つのHTMLファイルのみで動作)

動作環境

最新のWebブラウザ（Google Chrome, Microsoft Edge, Safari, Firefoxなど）で動作します。
インターネット接続が必要です（Tailwind CSSとフォントの読み込みのため）。
